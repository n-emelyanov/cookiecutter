# Cookiecutter Data Science

Cookiecutter Data Science (CCDS) - это инструмент для создания шаблона проекта в области DS, который включает в себя лучшие практики.

## Установка
```bash
# Установка через pip
pip install cookiecutter

# Или через pipx (рекомендуется)
pipx install cookiecutter
```

## Использование
```bash
cookiecutter https://github.com/n-emelyanov/cookiecutter
```

# Makefile
После формирования структуры папок выполните инициализацию проекта:
```bash
# Инициализация проекта (git + poetry)
make init
```
